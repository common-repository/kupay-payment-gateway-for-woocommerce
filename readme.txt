== Description ==

KuPay makes defi payments easy.

Tracks payments on KCC KuCoin Community chain and BSC Binance Smart Chain. Other chains coming soon!

Seamlessly integrates with MetaMask.

== Installation ==

Tested with WooCommerce 6.3 and above, might work on other versions.

For your API key, please sign up for a free Checkout account via https://kupay.finance/checkout

Installation guide:
1. Sign up for a new Merchant account on https://kupay.finance/checkout - takes one minute!
2. Then in the merchant panel you will find your API key on the Integrations page
3. Install the plugin in your WooCommerce website (search for "kupay kcc")
4. Enable the payment method
5. Enter your API key
6. Place a test order to check that everything goes well

More details here:
https://docs.kupay.finance/guides/setup-woocommerce
